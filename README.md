## IA em Produção — Pipeline RAG Local

Pipeline completo rodando em servidor próprio, integrando LLM local, banco vetorial e automação de fluxos:

```
[Telegram Bot] ──► [Whisper STT] ──► [n8n Automation]
                                             │
                                   [Ollama + Qwen2.5 LLM]
                                             │
                               [PostgreSQL + pgvector RAG]
                                             │
                          [OpenWebUI] ◄──────┘
                               │
                    [Groq / OpenRouter APIs]
```

| Componente | Tecnologia |
|---|---|
| LLM Local | Ollama + Qwen2.5 |
| Banco Vetorial | PostgreSQL + pgvector |
| Transcrição de Áudio | Whisper (speech-to-text) |
| Automação de Fluxos | n8n |
| Interface Web | OpenWebUI |
| APIs Externas de LLM | Groq, OpenRouter |
| Bot de Interação | Telegram Bot API |
| Monitoramento | Uptime-Kuma |
